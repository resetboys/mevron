# MEVRON

Стартовая структура для будущего сайта-визитки без React и без сборщика.

- `index.html` - разметка одностраничного сайта.
- `css/styles.css` - базовые стили проекта.
- `js/main.js` - файл для будущих скриптов.
- `assets/images/` - место для изображений, логотипов и экспортов из макета.

## Запуск

Откройте `index.html` в браузере. Сервер для текущей версии не нужен.
